这是一个显示在最近一天内发生的还原（restores）的快速脚本。


## English
Here’s a quick script which will show you restores that have taken place within the last day.

---
## Logic
```SQL
declare @within_last_day datetime
set @within_last_day = getdate() - 1 
-- finds restores within the last 24 hours select
-- 查找最近24小时内发生的还原（restores）
destination_database_name
, restore_date
from
msdb..restorehistory
where
restore_date = @within_last_day


```
